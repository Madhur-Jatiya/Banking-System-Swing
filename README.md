# Digital-Banking-System-Swing

"Digital Payment System" is a desktop application created using Swing and Java with database connectivity. It offers various services to the users, including sending money to someone using their username, adding money to their account, paying bills, and viewing transaction history. The application has a user-friendly interface designed using the Swing framework, providing an interactive experience for the users. With database connectivity, the application can store and manage user information, transaction details, and other necessary data, making it a powerful and efficient tool for digital payments. Overall, this application provides a seamless and convenient way for users to handle their financial transactions."
